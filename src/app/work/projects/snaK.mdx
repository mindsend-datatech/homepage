**Overview**

The snaK is a MetaMask Snap that enables native Kadena (KDA) blockchain functionality within MetaMask. It allows users to manage KDA accounts,
send and receive tokens, switch networks, and interact with Kadena-native dapps, all without having to install yet another extension in your
already bloated browser.

**Key Features**

- **Token and Account Management:** [Send KDA tokens](https://docs.mindsend.xyz/user-guide/send-kda) and self-custody multiple accounts across the chainweb;
- **Testnet Support:** [Access faucets](https://docs.mindsend.xyz/user-guide/faucet) for test tokens and test your dApps in open environments;
- **Full dapp support:** Interact with any Kadena dapp;
- **No extra packages:** No extra dependencies needed besides standard `window.ethereum` connection.

**Technologies Utilized**

- **MetaMask Snaps SDK:** Custom permissioned runtime inside MetaMask;
- **Kadena SDK (kadena.js):** Used to serialize/deserialize Pact code and handle signing workflows;
- **TypeScript, Nextjs, and Vercel:** snaK UI development and hosting;
- **Pact:** Kadena smart contract language powering chain interactions;
- **MdBook + GitHub Pages:** Documentation and developer portal.

**Challenges & Insights**

- **Cross-Chain Management:** Unified interaction across all 20 Kadena chains with on-the-go UI for cross-chain transaction finishing;
- **Snap Permission Constraints:** Account for Snap sandbox limitations to provide robust signing workflows;
- **dApp Compatibility:** Built extensible RPC handlers to support WalletConnect and custom dApp integrations.

**Achievements**

- **First MetaMask Integration for Kadena:** Provided access to Kadena's Chainweb for all Metamask userbase;
- **Seamless Developer Onboarding:** Offers clear [SDK setup](https://docs.mindsend.xyz/sdk/setup-project) and [Snap integration guides](https://docs.mindsend.xyz/sdk/integrate-snap) for dApps;
