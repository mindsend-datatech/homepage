**Overview**

The snaK wallet is a MetaMask Snap that enables native Kadena (KDA) blockchain functionality within MetaMask. It allows users to manage KDA accounts, send and receive tokens, switch networks, and access testnet tools — all seamlessly integrated within the MetaMask interface.

**Key Features**

- **Network Switching:** [Switch between Kadena networks](https://docs.mindsend.xyz/user-guide/switch-network)
- **Transaction Signing:** [Send KDA tokens](https://docs.mindsend.xyz/user-guide/send-kda) securely with native support for `sign` and `quicksign`
- **Testnet Support:** [Access faucets](https://docs.mindsend.xyz/user-guide/faucet) for test tokens and test your dApps in real environments
- **Permissionless Integration:** No plugin installation required beyond standard Snap connection
- **Full Account Management:** Interact with any Kadena chain and maintain full control over your public/private key pairs

**Technologies Utilized**

- **MetaMask Snaps SDK:** Custom permissioned runtime inside MetaMask
- **Kadena SDK (kadena.js):** Used to serialize/deserialize Pact code and handle signing workflows
- **TypeScript & React:** Frontend development and Snap configuration UI
- **Pact:** Kadena smart contract language powering chain interactions
- **MdBook + GitHub Pages:** Documentation and developer portal

**Challenges & Insights**

- **Cross-Chain Management:** Enabled flexible interaction across all 20 Kadena chains while maintaining signing integrity and transaction consistency
- **Snap Permission Constraints:** Navigated Snap sandbox limitations to enable robust signing workflows without compromising security
- **dApp Compatibility:** Built extensible RPC handlers to support WalletConnect and custom dApp integrations

**Achievements**

- **First MetaMask Integration for Kadena:** Introduced a fully functional Kadena wallet inside MetaMask for the first time
- **Seamless Developer Onboarding:** Offers clear [SDK setup](https://docs.mindsend.xyz/sdk/setup-project) and [Snap integration guides](https://docs.mindsend.xyz/sdk/integrate-snap) for dApps
- **Security-First Design:** Leverages MetaMask’s isolated execution model with strict permissions and validation
